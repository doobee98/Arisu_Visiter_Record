[Information]
	1. 숫자는 그냥 적어주시고, 문자열은 따옴표를 포함하여 적어주셔야 합니다.
	2. 일반 메모장으로 수정시 오류가 발생합니다. 인코딩이 다르기 때문입니다. 
	   인터넷에서 SublimeText3 등의 텍스트 에디터 프로그램을 다운받아서 수정하시기 바랍니다.


[Sample]
	1. 장소와 날짜는, 해당 내용을 입력할 셀의 위치에 location, date를 써 두시면
	   프로그램이 그 자리를 찾아서 입력해 줍니다.
	   location과 date를 쓰는 규칙이 있습니다.
	   (a) 셀 전체에 location이나 date를 작성
	   (b) 셀 일부분에 작성하려면 '(', ')' 괄호로 감쌀 것
	        - 예시: 'D1'셀에 '출입자 기록부 ( location )'이라 씀 -> '출입자 기록부 ( 암사 정문 )'으로 작성됨
	2. 엑셀에서 암호화파일로 옮길 시, 데이터베이스와 레코드를 구분하는 기준은
	   파일 이름에 <DB>가 들어갔는지 아닌지 입니다.
	   데이터베이스 암호화 파일로 변환하고 싶으면 파일 이름에 <DB>를 추가해 주시고,
	   기록부 암호화파일로 변환하고 싶으면 파일 이름에 <DB>가 포함되지 않도록 작성해 주세요.
